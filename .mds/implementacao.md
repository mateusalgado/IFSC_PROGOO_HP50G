# Implementação

O sistema foi implementado utilizando a interface de desenvolvimento integrada do `Qt Creator 17` que facilita a utilização da documentação e ecosistema do Qt.
O framework utilizado foi `Qt 6.5` para `C++` com `Widgets` e `CMake`.    
As bibliotecas Qt utilizadas foram `QtCore`, `QtWidgets`, `QtCharts`, `QtMqtt` e `QtSql`.  
O `Broker Mqtt` gratuito utilizado foi o [https://www.emqx.com/](https://www.emqx.com/).  
Os [ícones](/img/) utilizados foram baixados de [https://www.flaticon.com/free-icon/](https://www.flaticon.com/free-icon/).  
O `vscode` foi utilizado para o desenvolvimento e pré-visualização dos arquivos Markdown (.md).  
O [https://github.com/](https://github.com/) foi utilizado para o versionamento e armazenamento do código fonte e o projeto.  
O `Visual Paradigm 17.3` foi utilizado para elaboração dos arquivos UML.
