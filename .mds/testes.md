# Testes

Todas as funções implementadas no projeto foram testadas manualmente a partir da interação com o UI, incluindo a conexão com o broker mqtt, acesso ao banco de dados, exportação de dados e configuração do sistema.

Para o teste do broker mqtt foi desenvolvido um código simples em python para enviar dados aleatórios ao broker com uma frequência customizada.

Como resultado, todos os erros/bugs encontrados no SEMA foram corrigidos durante a implementação.
